# VegaStyle

VegaStyle est un site web de vente de vêtements, chaussures et accessoires avec système de commande et livraison.

---

##  Description

Projet de site e-commerce permettant la vente en ligne de :
- Chaussures
- Vêtements
- Accessoires

---

##  Fonctionnalités

- Affichage des produits
- Gestion des produits
- Contact client (WhatsApp ou formulaire)

---

##  Technologies utilisées

- HTML
- CSS
- JavaScript
